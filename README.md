# CRUD-API-Viacep

Umsistema CRUD (Create, Read, Update, Delete) para o cadastro de pessoas, integrado com a API ViaCep para obtenção de informações
de endereço através do CEP.

## Visão Geral

O CRUD-API-Viacep é uma aplicação que permite cadastrar informações de pessoas, incluindo nome, CPF, telefone, endereço e sexo. A
aplicação se destaca pela integração com a API ViaCep, que proporciona a obtenção automática de detalhes de endereço a partir do 
CEP fornecido.

Após cadastrar uma pessoa, você pode visualizar todos os registros cadastrados em uma tela separada.

## Funcionalidades

- Cadastro de pessoas com informações detalhadas.
- Integração com a API ViaCep para obtenção automática de endereço através do CEP.
- Visualização dos registros cadastrados.
- Edição e atualização de informações.
- Exclusão de registros.

## API ViaCep

A aplicação utiliza a API ViaCep para obter informações de endereço a partir do CEP fornecido. Isso é realizado através do CURL, que faz uma requisição à API e recebe os dados em formato JSON.
